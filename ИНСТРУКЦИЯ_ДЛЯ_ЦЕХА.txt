================================================================================
                  ИНСТРУКЦИЯ ДЛЯ ЦЕХА - DXF Stretcher
================================================================================

📦 ЧТО ЭТО?

DXF Stretcher - программа для автоматического растяжения разверток 
листового металла. Работает с DXF файлами.

================================================================================
🚀 КАК УСТАНОВИТЬ
================================================================================

1. Скопируйте файл DXF_Stretcher.exe на компьютер в цехе
2. Всё! Установка не требуется

⚠️  ВАЖНО: 
   - Python НЕ нужен
   - КОМПАС НЕ нужен (если работаете с DXF файлами)
   - Работает на любом Windows 7/10/11

================================================================================
📝 КАК ИСПОЛЬЗОВАТЬ
================================================================================

ВАРИАНТ 1: Обработка одной развертки
────────────────────────────────────────────────────────────────────────────

1. Запустите DXF_Stretcher.exe двойным кликом
2. Откроется окно программы
3. Перейдите на вкладку "Одна развертка"
4. Нажмите "Выбрать файл" - выберите DXF файл развертки
5. Нажмите "Выбрать основание" - выберите DXF файл основания
6. Нажмите "Измерить и растянуть"
7. Готово! Файл с суффиксом _stretch.dxf создан рядом с исходным

────────────────────────────────────────────────────────────────────────────

ВАРИАНТ 2: Пакетная обработка (несколько файлов сразу)
────────────────────────────────────────────────────────────────────────────

1. Запустите DXF_Stretcher.exe
2. Откройте вкладку "Пакетная обработка"
3. Нажмите "Выбрать папку" - укажите папку с файлами
   (В папке должны быть: основания + развертки)
4. Выберите эталон (внешний или внутренний радиус)
5. Выберите ось растяжения (X или Y)
6. Выберите привязку (начало/центр/конец)
7. Нажмите "Запустить обработку"
8. Дождитесь окончания (прогресс виден в окне)
9. Готово! Все файлы с _stretch.dxf созданы

────────────────────────────────────────────────────────────────────────────

ВАРИАНТ 3: Создание внешнего радиуса из внутреннего
────────────────────────────────────────────────────────────────────────────

1. Запустите DXF_Stretcher.exe
2. В главном меню выберите соответствующую опцию
3. Укажите файл внутреннего радиуса (DXF)
4. Укажите толщину металла (например, 1.5 мм)
5. Укажите радиус гибки (например, 3.0 мм)
6. Нажмите "Создать"
7. Готово! Файл внешнего радиуса создан

================================================================================
❓ ЧАСТО ЗАДАВАЕМЫЕ ВОПРОСЫ
================================================================================

Q: Программа не запускается
A: Проверьте, что это Windows 7 или новее. Антивирус может блокировать -
   добавьте программу в исключения.

Q: Где сохраняются результаты?
A: В той же папке, где находился исходный файл, с суффиксом _stretch.dxf

Q: Можно ли обработать файлы КОМПАС (.cdw, .frw)?
A: Нет, только DXF. Сначала экспортируйте из КОМПАС в DXF.

Q: Программа выдаёт ошибку при открытии файла
A: Убедитесь, что файл в формате DXF и не повреждён.

Q: Можно ли отменить растяжение?
A: Да, исходный файл не изменяется. Удалите файл *_stretch.dxf и 
   запустите обработку заново.

Q: Сколько файлов можно обработать за раз?
A: Неограниченно! Пакетная обработка обрабатывает все файлы в папке.
