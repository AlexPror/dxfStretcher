1. ЗАПУСК ПРОГРАММЫ
   • Дважды кликните на run.bat
   • Откроется окно программы

2. ПОДГОТОВКА ФАЙЛОВ
   • Поместите в одну папку:
     ✓ Файлы оснований: Основание Г1.корп1.dxf, Основание Г1.корп2.dxf, ...
     ✓ Файлы радиусов: Внешний радиус..., Внутренний радиус...

3. ПАКЕТНАЯ ОБРАБОТКА (рекомендуется)
   
   Вкладка "Пакетная обработка":
   
   ШАГ 1: Нажмите "Обзор" → выберите папку с файлами
   
   ШАГ 2: Нажмите "1. Анализировать основания"
           • Программа найдёт все основания
           • Извлечёт дуги и их длины
           • Покажет список файлов для обработки
   
   ШАГ 3: Установите параметры:
           • Направление: X (обычно)
           • Центр: Левый край (обычно)
   
   ШАГ 4: Нажмите "2. Запустить обработку"
           • Программа автоматически обработает все файлы
           • Целевые длины определяются из оснований
   
   ШАГ 5: Нажмите "3. Создать отчёт"
           • Будет создан файл ОТЧЁТ_ОБРАБОТКА_*.txt

4. РЕЗУЛЬТАТ
   • Файлы с суффиксом _stretch.dxf
   • Текстовый отчёт с детальной информацией

   • Все ли файлы успешно обработаны?
   • Правильные ли операции (удлинение/укорочение)?

2. Проверьте ВЫХОДНЫЕ ФАЙЛЫ:
   • *_stretch.dxf в той же папке
   • Откройте в CAD и проверьте размеры

3. Проверьте ОТЧЁТ:
   • Полная информация о каждой обработке
   • Использованные основания и дуги
   • Коэффициенты и изменения

ОБЯЗАТЕЛЬНО должно быть слово "корп" + номер:

  ✓ Основание Г1.корп1 - 1шт.dxf
  ✓ Внешний радиус Г.1корп1 - 1шт.dxf
  ✓ Внутренний радиус Г1.корп2 - 1шт.dxf
  ✗ Основание Г1.1 - 1шт.dxf (нет "корп")


┌──────────────────────────────────────────────────────────────────────────────┐
│ УСТРАНЕНИЕ ПРОБЛЕМ                                                           │
└──────────────────────────────────────────────────────────────────────────────┘

"Не найдено файлов оснований"
  → Проверьте, что файлы называются "Основание*.dxf"

"Не найдено основание для корпХ"
  → Убедитесь, что есть файл "Основание ... корпХ ..."
  → Проверьте правильность написания "корп"

"Найдено менее 2 дуг"
  → Откройте файл основания в CAD
  → Убедитесь, что есть минимум 2 дуги (ARC)
